Java中继承无参数构造函数出错

Posted on 2017-12-23 Edited on 2021-03-13

今天用eclipse继承一个函数之后，报了个错误Implicit super constructor Test() is undefined for default constructor. Must define an explicit constructor。刚开始很纳闷，没问题啊，为啥会报错，后面去网上查了下发现了原因：

如果继承的父类里只有含参数的构造函数，编译器不会默认为你生成一个无参数的构造函数。
当你的子类继承的时候，只有在自己的构造函数中显示调用父类的构造函数，才能确保子类在初始化之前父类会被实例化
