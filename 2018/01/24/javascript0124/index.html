    javascript基本用法

    • javascript可以在html中内嵌语句，也可以调用外部的.js文件(可以在heard中也可以在body中)。
    • javacaript类似于java由多条语句组成，语句结束用分号表示。
    • JavaScript的注释也同java，单行//，代码块/* */
    • JavaScript变量 var 变量名这样定义
    • JavaScript函数 function 函数名(){}定义 , 函数名()调用

    JavaScript输出内容

    document.write()是直接指向HTML的输出流写内容。
    如果需要输出空格有以下两种方法：

    • 1.使用输出html标签&nbsp;来解决，例document.write("&nbsp;&nbsp;"+"1"+"&nbsp;&nbsp;&nbsp;&nbsp;"+"23");
    • 2.使用css样式来解决，在输出时添加white-space:pre样式属性，例document.write("<span style='white-space:pre;'>"+" 1 2 3 "+"</span>");

    JavaScript消息对话框

    网页中弹出的小窗口就是JavaScript的消息对话框实现的

    第一种消息对话框，就是alert()方法，弹出对话框。
    第二种消息对话框，当弹出对话框时，有确认和取消两个选项，也就是confirm()。返回类型是boolean类型
    第三种消息对话框是提问类型prompt(str1,str2)

    参数：str1显示消息对话框的文本(不可修改)，str2文本框的内容(可修改)
    返回值：点确定，文本框的内容将作为函数的返回值。点取消返回null

    JavaScript打开窗口

    语法

    window.open([URL],[窗口名称],[参数字符串])

    参数说明：

    URL：可选参数，在窗口中显示网页或路径，如果省略这个参数，或者值为空，则窗口不显示任何文档。

    窗口名称： 可选参数，被打开窗口的名称。

    • 该名称由字母，数字，下划线组成
    • “_top”,”_blank”,”_self”具有特殊意义的名称。
      _blank：在新窗口显示目标网页
      _self：在当前窗口显示目标网页
      _top：框架网页中在上部窗口中显示目标网页
    • 相同的name的窗口只能创建一个
    • name不能含有空格
      参数字符串：可选参数，设置窗口参数，各个参数逗号隔开.

    语法：

    window.close()（关闭本窗口）或<窗口对象>.close

    用法：

    将新打开窗口的对象存储在变量中，用变量调用变量.close()

    通过id获取元素

    语法：

    `document.getElementById(“id”);

    用法：

    可以将获取的元素储存在变量中，随后使用变量，例如：var element = document.getElementById("id"); doucument.write(element)
    ps:document.getElementById(“id”);得到的是id的对象。document.getElementById(“id”).innerHTML得到的才是元素内容;
